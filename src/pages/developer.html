The site is composed mostly of its website code, and graphs and charts hosted on Tableau Public.

The Website Code

The website code is on github. There is a DOCUMENTATION.md markdown file that describes the project (in addition to basic setup in the readme.md)

  • You could edit the html files in the ftp site directly, but I recommend grabbing this project and using the build process instead - you'll have less work and end up with more awesome.
  • The readme.md in the project details the basucs of the ZURB template and how to get it installed.
  • The working.md file is a quick reminder of how to get the project going and how to update it as you go. :)

The Graphs

The graphs all stream in from Tableau Public. You'll need a licensed copy of Tableau Desktop to update them. You can find the Tableau Profile that owns the workbook.

The site access information, like ftp access, the accounts for hosted videos/etc., are held by MCA and Warren & Co .
